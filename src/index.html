The following is the task of which produced this website:

Each group must produce a webpage (running as a server) that shows a map of Norway
with appropriate symbols for the predicted weather at major cities.
Hint use https://api.met.no , Google Maps and possibly fontawesome.com .
The system needs only show a forecast for "Today" and "Tomorrow".
The user must be able to browse locations in Norway and the map must reflect this.
This task is fairly open ended, so I'm leaving it to individual groups to be creative.
Hand in a complete archive (.war) that I can launch on my own computer.
